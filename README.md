# Personal Portfolio

A modern, responsive personal portfolio website showcasing my projects, research, technical skills, and experience.

## About

This portfolio highlights my work in Artificial Intelligence, Machine Learning, and Software Engineering. It features my projects, research experience, technical skills, education, and contact information.

- Responsive design for desktop, tablet and mobile
- Both Light and Dark theme option
- Animated hero section
- Skills showcase
- Experience section
- Featured projects
- Contact section with social links
